# Kullanıcı Bilgi Kartı
### Bu proje, Python programlama dilinde temel veri tiplerini kullanarak bir kullanıcının kişisel bilgilerini saklayan ve bu bilgileri formatlanmış bir şekilde terminale yazdıran bir başlangıç uygulamasıdır.

## Ne Yapıyor?
* Farklı veri türlerindeki değişkenleri (Metin, Sayı, Mantıksal Değer) tanımlar.

* Değişkenleri dinamik olarak birleştirir (Concatenation).

* Veri tipi dönüşümü (Type Casting) yaparak terminale temiz bir çıktı verir.

## Hangi Konular Var?
* Veri Tipleri: String, Integer, Boolean.

* Tip Dönüşümü: str() fonksiyonu kullanımı.

* Çıktı Formatlama: Escape characters (\n) ile alt satıra geçme ve metin birleştirme.

## Nasıl Çalıştırılır?
* Bilgisayarınızda Python yüklü olduğundan emin olun.

* Proje dosyasını (main.py veya kullanici.py) terminalde açın.

* Aşağıdaki komutu çalıştırın:

```Bash
python dosya_adi.py
```
